
commit : Depodaki değişiklikleri kaydedin
Kaynak : https://www.git-scm.com/docs/git-commit

Yapılan Değişiklikler add Yabdıkdan Sonra Kayıt İşlemini Gerçeklesir
commit Eterken -m (mesaj) Eklemessen Tanıtılmıs IDE ile Girmeni isdiyor

$ git commit -m "Commit Mesajı" // Commit işlemi Gerceklesir Ve Birde Mesaj Ekler
$ git commit -am "Commit Mesajı" // add Yapıp Mesajla Commit Etiyor
$ git commit -a // add yapıp commit etiyor
